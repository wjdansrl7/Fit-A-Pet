##  🎥 프로젝트 개요
> 헬스 데이터를 활용한 펫 성장 서비스

> 코로나 19 이후 건강관리 시장 급성장<br/> 건강관리 주소비사로 부상한 MZ 세대<br/> 새로운 트렌드의 흐름 속에서 풍족한 헬스플레저 경험을 위한 하나의 컨텐츠를 제공

## 사용 방법
### 🔐 Backend
> ```
> $ ./gradlew clean build;
> $ java -jar build/libs/*SNAPSHOT.jar
> ```
### 🌅 Front-end
> ```
> $ npm install
> $ npm start
> ```
