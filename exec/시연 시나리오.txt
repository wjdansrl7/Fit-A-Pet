카카오 로그인
1. Fit-A-Pet 애플리케이션 클릭
2. [카카로오 시작하기] 클릭
3. 카카오 계정으로 로그인하고 [continue] 클릭
4. 처음 로그인하면 알이 생성되는데, 펫의 이름을 설정하거나 기본 이름 사용

메인 화면
1. 상단 펫 이름 옆 펜 버튼 클릭 시, 펫의 이름 변경 가능

- 식단기록 화면
1. 우측의 [식단기록] 클릭하면 카메라가 켜지고 음식 사진을 찍으면 영양성분이 로드되고 기록에 등록

- 퀘스트 화면
1. 우측의 [퀘스트] 클릭하면 퀘스트 화면으로 넘어가고 개인 퀘스트와 그룹에서 설정된 퀘스트 목록 확인 가능
2. 퀘스트 완료 시 complete 표시와 함께 경험치 상승, 레벨 업 달성 시, 펫 진화(Lv10 당)와 알 지급(Lv30 이상)

- 길드(그룹) 화면
1. 하단의 [지도] 클릭하면 지도 화면으로 넘어가고 가입한 길드 목록 조회 가능
2. 빈 구역을 클릭하면 길드를 생성하거나 생성돼있는 길드에 참여 가능
3. 길드장이라면 길드 퀘스트 등록 가능 -> 길드원이라면 메인의 [퀘스트]에서 이를 확인 가능
비어있는 길드원 칸을 클릭하면 참여코드를 발급 -> 다른 유저가 입력하고 그룹에 참여 가능 

- 도감 화면
1. 하단의 [도감] 클릭하면 도감 화면으로 넘어가고 보유한 펫 목록 조회 가능
2. 펫을 클릭하면 진화 단계에 따른 모습과 상세정보 조회 가능
3. 상세화면에서 메인 펫 변경 가능

- 기록 화면
1. 하단의 [기록보기] 클릭하면 나의 기록 화면으로 넘어가고 일일 섭취량, 수면 시간, 걸음 수 확인 가능
2. 일일 섭취량 클릭 시 오늘 섭취량 확인 가능
3. [로그아웃] 버튼 클릭 시 로그아웃
